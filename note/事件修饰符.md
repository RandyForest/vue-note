# 事件修饰符

## 用法

```html
<标签 v-on:事件.修饰符="方法"></标签>
```

可以串联使用

```html
<标签 v-on:事件.修饰符1.修饰符2="方法"></标签>
```



## 修饰符

| 一般事件修饰符 | 说明                                                        |
| -------------- | ----------------------------------------------------------- |
| `.stop`        | 阻止事件继续传播                                            |
| `.prevent`     | 事件不再重载页面                                            |
| `.capture`     | 先在自定义方法中处理，然后才交由内部元素进行处理            |
| `.self`        | 只当在 `event.target` 是当前元素自身时触发处理函数          |
| `.once`        | 只触发一次                                                  |
| `.passive`     | 立即触发默认行为，不等待自定义方法执行完成。会覆盖`prevent` |

| 按键修饰符 | 说明           |
| ---------- | -------------- |
| `.enter`   | 回车键         |
| `.tab`     | Tab键          |
| `.delete`  | 删除键与退格键 |
| `.esc`     | 退出键         |
| `.space`   | 空格键         |
| ...        |                |

| 系统按键修饰符 | 说明 |
| -------------- | ---- |
| `.ctrl`        |      |
| `.alt`         |      |
| `.shift`       |      |
| `.meta`        |      |



| 鼠标按钮修饰符 | 说明 |
| -------------- | ---- |
| `.left`        | 左键 |
| `.right`       | 右键 |
| `.middle`      | 中键 |



| 其他事件修饰符 | 说明         |
| -------------- | ------------ |
| `.exact`       | 精准触发事件 |
